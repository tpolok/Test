Qu'est ce qu'une PWA ?

Definition

PWA est une abbréviation pour "Progressive Web App", on pourrait croire au premier abord que ce site est un site banal.

Cependant, la particularité de ce dernier est, qu'une fois accédé via un appareil mobile sous Android ou iOS, un message apparait nous demandant si l'on désire installer le site sur notre appareil

Si vous acceptez, le site viendra s'installer sur votre appareil et se disponible de la même manière qu'une application téléchargé sur le store de votre appareil !

Ce site sera disponible hors ligne et stocké dans la mémoire de votre appareil ! Le rendant beaucoup accessible si vous consultez régulièrement !

Histoire

Le terme PWA est assez jeune puisqu'il date de 2015.

C'est un ingénieur de Google qui proposa ce terme pour désigner ce genre de site web

Il existait donc des PWA avant même que le terme existe, cependant, elles étaient très rares et la démocratisation se fit une fois la chose nommée

Les promesses d'une PWA

Si une PWA est construite correctement, alors, par définition, elle promet les choses suivantes :

  • Elle est progressive, elle fonctionne peu importe le navigateur
  • Elle est responsive, le design s'adapte selons la taille de l'appareil
  • Elle fonctionne hors ligne, le service worker le permettant, ce qui sera expliqué plus tard.
  • Elle ressemble et à les codes d'une application mobile
  • Elle est toujours à jour et demande à se mettre à jour une fois qu'une connexion internet est disponible et qu'une nouvelle version est disponible
  • Elle est "sure", elle est servie en HTTPS
  • Elle est engagente avec des push notifications etc..
  • Elle peut s'installer
  • Facilement partageable, le lien d'index suffit

La plupart de ces caractéristiques sont vérifiables avec Google Chrome, ce dernier possédant un outil vérifiant si le site est bien une PWA

Pour y accéder :

  • Ouvrez Google Chrome
  • Rendez vous sur le site à tester
  • Appuyez sur la touche F12 de votre clavier
  • Rendez vous dans l'onglet "Audit"
  • Cliquez sur "Lancer un audit"

Après tout cela, votre navigateur lancera une série de tests, et vous communiquera les résultats du site quelques instants plus tard
